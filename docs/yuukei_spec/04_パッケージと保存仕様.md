# Yuukei 完成版実装仕様書 v1.2
## 04. パッケージと保存仕様

## 10. パッケージ仕様

### 10.1 パッケージ定義

パッケージは次の要素のうち 2 つ以上を含む配布単位。

- VRM
- 台本
- モーション
- テクスチャ
- 小物
- DLL

### 10.2 役割

- パッケージは世界観の一括適用単位
- 現在有効パッケージが外見と振る舞いの既定値を提供する

### 10.3 個別差し替え

- パッケージ適用後、各構成要素を個別差し替えできる
- 個別差し替えはパッケージに対する上書きとして扱う

### 10.4 パッケージ切替時

- 既存の個別差し替えは引き継がない
- 新しいパッケージの既定構成から再スタートする

### 10.5 パッケージ破損時

- 壊れている要素のみ無効化して継続する
- ロード時に警告を出す
- アプリ全体停止やパッケージ全体拒否は既定動作にしない

### 10.6 パッケージ ID

- GUID を使用する

### 10.7 保存先

```text
Application.persistentDataPath/package/{creator}-{version}-{guid}/
```

### 10.8 予約ファイル名

- キャラクターモデル本体: `character.vrm`
- マニフェスト名: `manifest.json`

### 10.9 破損判定

- ロードに失敗したかどうかを基準とする

### 10.10 manifest.json 形式

```json
{
  "creator": "hoge",
  "version": "v0.0.1",
  "download": "https://example.com/latest",
  "license": "hoge",
  "id": "00000000-0000-0000-0000-000000000000",
  "character": "character.vrm",
  "daihon": [
    "Scripts/バレンタイン.daihon",
    "Scripts/正月.daihon"
  ],
  "textures": {
    "speechBubble": {
      "background": "Textures/speech_bubble_bg.png",
      "tail": "Textures/speech_bubble_tail.png"
    }
  },
  "assets": [
    "Props/item_01.prefab",
    "Props/item_02.prefab"
  ],
  "motions": [
    "Motions/wave.vrma",
    "Motions/idle.vrma"
  ],
  "dlls": [
    "Plugins/example.dll"
  ],
  "aliases": {
    "events": {
      "起動時": "app_started",
      "クリック": "character_clicked",
      "ダブルクリック": "character_double_clicked",
      "ドラッグ開始": "character_drag_started",
      "ドラッグ終了": "character_drag_ended",
      "ファイルドロップ": "file_dropped",
      "放置": "idle_reached",
      "定期発火": "periodic_tick"
    },
    "functions": {
      "吹き出し表示": "show_dialog",
      "表情変更": "set_expression",
      "モーション再生": "play_motion",
      "小物表示": "set_prop_visible",
      "選択肢表示": "show_choices",
      "永続保存": "set_persistent"
    }
  }
}
```

### 10.11 マニフェスト解釈規則

- すべてのパスはパッケージルートからの相対パス
- `character` は単数
- `daihon` は順序付き配列
- `daihon` は配列順にロードする
- `assets` / `motions` / `dlls` は配列
- `textures` は名前付き辞書
- `download` はメタ情報
- `daihon` の各要素は相対パスである
- 壊れた `daihon` は個別に警告してスキップする
- `dlls` は存在しても自動ロードしない
- `aliases` は省略可能
- `aliases.events` / `aliases.functions` は **alias -> canonical** の辞書
- パッケージ未定義の alias はアプリ組み込み alias を使用する

## 11. 保存仕様

### 11.1 保存先

- 一般状態: `Application.persistentDataPath/save.json`
- API キー: OS 提供のセキュアストア

### 11.2 save.json 形式

```json
{
  "activePackageId": "00000000-0000-0000-0000-000000000000",
  "overrides": {
    "daihon": [],
    "character": "",
    "textures": {},
    "assets": {},
    "motions": {}
  },
  "persistentVariables": {
    "valName": "valValue"
  },
  "appState": {
    "isTemporarilyDisabled": false,
    "isTemporarilyHidden": false,
    "shortcutConfig": {
      "toggleDisabled": "Ctrl+;",
      "toggleHidden": "Ctrl+Shift+;"
    }
  }
}
```

### 11.3 保存ルール

- `persistentVariables` は Daihon 永続変数の実体
- `overrides` は現在有効パッケージへの上書きのみ保持
- `overrides.daihon` は有効 Daihon 配列全体への上書きとして扱う
- `overrides.daihon` が空配列のときはパッケージ既定の Daihon 配列を使用する
- パッケージ切替時は `overrides` を初期化
- 一時変数は保存しない
- キャラクター位置は保存しない
- 実行中選択肢状態、吹き出し表示状態、再生中モーション状態は保存しない

### 11.4 型ルール

- `persistentVariables` に保存する型は 真偽値 / 数値 / 文字列
- JSON では通常の JSON 型へマッピング
- 読み込み時に型不整合があればその変数を無視し警告する
