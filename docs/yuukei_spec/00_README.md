## 分割方針

- **読んで理解するための仕様** と **そのまま実装依頼に使う指示** を分離しました。
- 実装中に頻繁に見るものを独立させるため、**Daihon 実行契約** を単独ファイルにしました。
- UI、保存、パッケージ、実装責務を別ファイルにして、修正時の差分管理がしやすい形にしています。
